# Markdown Cheat Sheet


### Format Text

```markdown
# This is an <h1> tag
## This is an <h2> tag
### This is an <h3> tag
#### This is an <h4> tag
##### This is an <h5> tag
###### This is an <h6> tag
```
# This is an H1 tag
## This is an H2 tag
### This is an H3 tag
#### This is an H4 tag
##### This is an H5 tag
###### This is an H6 tag
####### This is an H7 tag

```markdown
*This text will be italic*
_This will also be italic_
**This text will be bold**
__This will also be bold__

*You **can** combine them*
```

### List

```markdown
* Item 1
* Item 2
  * Item 2a
  * Item 2b
```

```markdown
1. Item 1
2. Item 2
3. Item 3
   * Item 3a
   * Item 3b
```
